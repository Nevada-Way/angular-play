Info for developers :

The youtube URL in IFrame only works if urls are of type "embed".

You get it using the "share" button on YouTube, then selecting "embed" option.

If you have a regular youtube link then you need to replace this: "/watch?v=" with this: "/embed/".

Regarding the input box below, if the url is of type "watch" then it auto converts to "embed" when activating the IFrame, the display in the input box doesnt change.
